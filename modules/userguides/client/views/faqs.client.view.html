ไม่สามารถอัพโหลดโปรแกรมไปที่ Arduino ได้

ลองตรวจสอบว่าได้ติดตั้งไลบรารี่ "Altsoftserial" ไว้แล้วหรือไม่ หากยังไม่ได้ติดตั้งลองทำตามขั้นตอนดังต่อไปนี้

เปิด arduino IDE → sketch → Manage Libraries → and search "Altsoftserial" and install

จำนวนข้อมูลสูงสุดที่สามารถส่งได้สำหรับแมคเจลเลน

หน่วยความจำภายใน (dynamic memory) ของ Arduino มีผลกับจำนวนตัวอักษรของ payload หรือ จำนวนข้อความที่ส่งขึ่นไปที่แมคเจลเลน

เนื่องจากบางบอร์ดมีหน่วยความจำ (dynamic memory) ค่อนข้างน้อย เช่น arduino UNO ดังนั้น การใช้งานจึงอาจมีข้อจำกัดเมื่อต้องการส่งข้อความเป็นจำนวนมาก แนะนำให้ใช้บอร์ดรุ่นอื่นๆ เพื่อประสิทธิภาพและความยืดหยุ่นในการส่งข้อมูล

หมายเหตุ จากการทดสอบกับ example post.ino สามารถส่งข้อมูลได้สูงสุด 100 ตัวอักษรของการส่งข้อมูลต่อครั้ง

Serial monitor ใน Arduino โชว์ "connecting........ >> Create Socket" แล้วค้าง

ให้ลองตรวจสอบโปรแกรมดูว่า ในฟังก์ชั่น SETUP มีการใช้คำสั่งใดๆ ที่ทำให้ไม่สามารถ เข้าในฟังก์ชั่น Loop ได้หรือไม่ เนื่องจากบางครั้งมีโค้ดมีการวนซ้ำ (Loop) เพื่อเช็คอะไรบางอย่าง เช่น การเชื่อมต่อของเซ็นเซอร์ อุณหภูมิแบบ l2C ซึ่งเป็นผลให้ไม่สามารถทำงานต่อในฟังก์ชั่นอื่นๆ ได้

หมายเหตุ เพื่อเป็นการพิสูจน์ข้อเท็จจริง แนะนำให้ลองทดสอบกับ example ของ Magellan โดยอาจจะลองใช้ "post.ino" ในการทดสอบ
