Get Start Magellan

The Magellan library use for develop with NB-IoT. This use CoAP protocol

  • - post method
  • - get method

INSTALLATION

  • 1. Arduino IDE Download > Download
  • 2. Download library Magellan > Magellan Library
  • 3. Include library in Arduino IDE
  • 4. Open Magellan Example "POST_MAGELLAN"

Create Things

First step to have a Thing proxy for connecting Things to Magellan

  • 1. Goto Things Page and click on the button "Create Thing".
  • 2. Fill in the form.

Embed The Key

Introduce your Things to Magellan

  • 1. Click on your Thing Box in page All Things
  • 2. Click "Copy" button, then the Token Key will be copied in the clipboard
  • 3. Paste the Token Key in POST_MAGELLAN and click Upload
  • 4. You can Monitor Data Flow on thing detail in Magellan Platform or Arduino IDE Serial Monitor

Create Dashboard

To create a spreadsheet where you can monitor or interact with your Things

  • 1. Goto Dashboard Page and click on the button "Create Dashboard".
  • 2. Fill in the form, "Dashboard Name" must not be empty. if you have no other information. you can come back and edit later

Create Widget

Widget is a controller or a monitor for your Things

  • 1. Goto Dashboard Page and click on the button "Create Widget".
  • 2. Please ensure that you have created a Thing proxy and embeded Token Key. Check the connection between Thing and Magellan must show "Connected".
  • 3. if you have created a widget and connected it with your thing. you can Visualize Your Data in the dashboard

Create Group

Manage your Things in a group

  • 1. Goto Group Page and click on the button "Create Group".
  • 2. Fill in the form.
  • 3. Then, go in the group you have created and Add Things into the Group.

HTTP API

You can use HTTP REST APIs to get your things data from Magellan to other application, clouds, or third party platform

  • 1. The link to the data is shown in HTTP API, which you can copy the link to use in your application or the third party platform.
  • 2. Example data in json format.
  • 3. For example, you could use dweet.io or Freeboard to display and control your data.
